## 功能模块

### 用户功能

- **语音垃圾分类**：用户上传音频文件，系统通过语音识别自动判断垃圾分类。
- **个人中心**：用户可查看和修改个人信息，以及历史分类记录。
- **注册与登录**：用户需注册并登录后使用系统功能。

### 管理员功能

- **分类规则维护**：管理员可对垃圾分类规则进行新增、修改和删除。
- **用户管理**：管理员负责用户信息的查看、编辑和删除。
- **系统信息管理**：管理员可更新首页信息和公告等内容。

### 扩展功能

- **分类结果追溯**：每次分类操作都会记录详情，便于查询。
- **多语言支持**：语音识别功能支持多种语言，拓展了系统的适用范围。

## 系统角色

- **用户**：使用系统进行垃圾分类的普通用户。
- **管理员**：负责系统维护和用户管理的角色。
